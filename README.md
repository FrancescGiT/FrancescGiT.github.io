# Perfil interactivo de Francesc Benach Garcia

Portfolio profesional interactivo centrado en ingeniería electrónica, dirección de proyectos, transformación digital e inteligencia artificial aplicada.

## Desarrollo local

```bash
npm install
npm run dev
```

## Publicación

El repositorio incluye un flujo de GitHub Actions que construye la aplicación y la publica automáticamente en GitHub Pages cuando se actualiza la rama `main`.

En **Settings → Pages → Build and deployment**, selecciona **GitHub Actions** como origen de publicación.
